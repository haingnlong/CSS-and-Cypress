Phân biệt class và id

class

  • Cocacola
  • Pepsi
  • 7 Up
  • Spite
  • Fanta
Dùng hàm tìm kiếm ra được:

1 class có thể dùng cho nhiều phần tử và ta có thể tìm ra tất cả các phần tử đó.

id

  • Cocacola
  • Pepsi
  • 7 Up
  • Spite
  • Fanta
Dùng hàm tìm kiếm ra được:

1 id chỉ có thể dùng cho 1 phần tử duy nhất, nếu dùng cho nhiều phần tử thì cũng không hỗ trợ tìm các phần tử khác ngoài phần tử đầu tiên sử dụng.

First Child (:first-child)

Sử dụng CSS Combinator để hoàn thành mong muốn:

File HTML:
<ul>
<li class="item">Item1</li>
<li class="item">Item2</li>
<li class="item">Item3</li>
<li class="item">Item4</li>
<li class="item">Item5</li>
</ul>
Mong muốn:
File CSS:
.item {
list-style: none;
margin-top: 5px;
}
{
border-top: 1px solid grey;
}
Thực tế:
  • Item1
  • Item2
  • Item3
  • Item4
  • Item5
